表面被覆率 Θ(t) シミュレーター - SI物理モデル版

吸着、Polanyi-Wigner 型の熱脱離、イオン誘起脱離を、表面サイト密度で規格化した 0D 平均場モデルとして扱います。 通常はプラズマ状態パラメータだけを操作し、材料・モデル詳細は必要なときだけ開いて調整します。

プラズマ状態パラメータ
単位: m^-2 s^-1。反応性中性種の表面入射フラックス。
単位: K。Polanyi-Wigner 項の Arrhenius 因子を通じて熱脱離に効きます。
単位: m^-3。Γi = n_se sqrt(e Te / mi) によりイオンフラックスを定めます。
単位: eV。Bohm 速度 uB = sqrt(e Te / mi) に効きます。
単位: eV。単一代表エネルギーでイオン誘起脱離収率を与えます。
表面・材料パラメータ
単位: m^-2。1 ML のサイト数。典型値は 1e19 m^-2 程度。
無次元。空きサイトに到達した中性種が吸着する有効確率。
単位: eV。coverage 依存を切れば一定障壁として使います。
単位: s^-1。n = 1 の既定では一次脱離の典型オーダーです。
イオン誘起脱離モデル
単位: amu。Ar+ なら約 40 amu、He+ なら約 4 amu。
単位: eV。Ei が Eth 以下なら Y(Ei) = 0 とします。
単位: adsorbate / ion。実材料では較正が必要です。
Polanyi-Wigner 詳細設定
既定は n = 1。n = 1 かつ coverage 依存なしでは厳密更新を使います。
単位: eV / coverage。Edes_eff = Edes,0 + aE (Θ - 0.5)。
無次元。ν_eff = ν0 exp[aν (Θ - 0.5)] の形で補正します。
表示設定
既定は 0.1x。物理式は変えず、画面上の進行速度だけを変えます。
単位: s。短時間の立ち上がりを見るなら 1 から 5 s が見やすいです。
ODE には影響しません。ラジカルやイオンが降る見た目だけを調整します。
既定では非表示です。吸着を正、熱脱離とイオン誘起脱離を負として別パネルに出します。

注意: このモデルは 0D の平均場モデルです。表面拡散、サイト不均一の詳細分布、角度分布、IEDF、readsorption、 ラジカル再結合、RF シース波形は含んでいません。通常モードでは一次 Polanyi-Wigner 脱離を仮定し、 n = 1 かつ coverage 依存を切った場合に限って厳密更新を使います。実験に合わせるには Ns、s、Edes、ν、Y(Ei) を較正してください。
